\f0\fs24 \cf0 # Exploratory Data Analysis of Highest Grossing Films\
\
## Overview\
\
This project was completed as part of the CodeAlpha Data Analytics Internship.\
\
The objective of this task is to perform Exploratory Data Analysis (EDA) on a dataset of the world's highest-grossing films. The analysis focuses on understanding the structure of the data, identifying missing values and duplicates, cleaning the dataset, and extracting meaningful insights.\
\
## Features\
\
- Loaded and explored the dataset\
- Checked data types and dataset structure\
- Identified missing values and duplicate records\
- Cleaned the Worldwide Gross column\
- Generated descriptive statistics\
- Analyzed movie release years\
- Identified top highest-grossing films\
\
## Technologies Used\
\
- Python\
- Pandas\
- Jupyter Notebook / Google Colab\
\
