# Exploratory Data Analysis of Highest Grossing Films

## Overview

This project was completed as part of the CodeAlpha Data Analytics Internship (Task 2: Exploratory Data Analysis).

The objective of this task is to analyze the dataset of the world's highest-grossing films and uncover meaningful insights through data exploration and statistical analysis. The dataset was examined for its structure, cleaned where necessary, and analyzed to understand trends and patterns.

## Objectives

- Understand the structure of the dataset
- Check for missing values and duplicate records
- Clean and prepare the data for analysis
- Generate descriptive statistics
- Explore release year trends and worldwide gross values
- Identify key observations from the dataset

## Dataset

The dataset contains information about the highest-grossing films collected from Wikipedia.

The main attributes include:

- Rank
- Peak Position
- Title
- Worldwide Gross
- Year

## Analysis Performed

- Loaded and inspected the dataset
- Verified data types and dataset dimensions
- Checked for missing values
- Checked for duplicate records
- Converted the "Worldwide Gross" column into numeric format
- Generated summary statistics
- Analyzed release years and top-grossing movies

## Technologies Used

- Python
- Pandas
- Jupyter Notebook / Google Colab

## Files Included

- `Task2_EDA.ipynb` – Complete exploratory data analysis notebook
- `highest_grossing_films.csv` – Dataset used for analysis
- `requirements.txt` – Required Python packages

## Key Findings

- The dataset contains structured information about the world's highest-grossing films.
- Most films in the dataset belong to recent decades.
- Worldwide gross values vary significantly across movies.
- Data cleaning improved the quality of the dataset for further analysis.

   ```

   ```

4. Review the generated outputs and insights.
