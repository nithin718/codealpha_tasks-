# CodeAlpha Data Analytics Internship

## Overview

This repository contains the projects completed as part of the CodeAlpha Data Analytics Internship.

### Task 1 – Web Scraping
- Extracted the highest-grossing films dataset from Wikipedia.
- Cleaned and exported the data as a CSV file.

### Task 2 – Exploratory Data Analysis (EDA)
- Performed data cleaning and statistical analysis.
- Identified trends, missing values, and key insights.

### Task 3 – Data Visualization
- Created visualizations using Matplotlib.
- Analyzed revenue distribution, release years, and top-grossing films.

## Technologies Used

- Python
- Pandas
- Requests
- Matplotlib
- Jupyter Notebook
