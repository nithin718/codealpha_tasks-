# Data Visualization of Highest Grossing Films

## Overview

This project was completed as part of the CodeAlpha Data Analytics Internship (Task 3: Data Visualization).

The objective of this task is to transform raw movie data into meaningful visual representations using Python and Matplotlib. The visualizations help identify trends, compare movie revenues, and understand the distribution of highest-grossing films across different years and decades.

## Objectives

- Create meaningful visual representations of the dataset
- Compare worldwide gross earnings of top films
- Analyze movie release patterns over time
- Understand the distribution of worldwide gross values
- Present insights through charts and graphs

## Dataset

The dataset consists of information about the world's highest-grossing films extracted from Wikipedia.

The dataset includes:

- Rank
- Peak Position
- Title
- Worldwide Gross
- Year

## Visualizations Created

### Top 10 Highest Grossing Films
A bar chart comparing the worldwide earnings of the ten highest-grossing movies.

### Movies Released by Year
A line chart showing the number of movies released across different years.

### Distribution of Worldwide Gross
A histogram illustrating how worldwide earnings are distributed among the films.

### Movies by Decade
A pie chart representing the proportion of highest-grossing films released in each decade.

## Technologies Used

- Pandas
- Matplotlib
- Jupyter Notebook / Google Colab

## Files Included

- `Task3_DataVisualization.ipynb` – Complete visualization notebook
- `highest_grossing_films.csv` – Dataset used for visualization
- `requirements.txt` – Required Python packages

## Key Insights

- Avatar remains the highest-grossing film in the dataset.
- Modern films dominate the list of highest-grossing movies.
- Worldwide gross values are unevenly distributed, with only a few films earning exceptionally high revenue.
- The 2000s and 2010s account for a large share of the highest-grossing films.
